```
usage: conkat_seq (--pacbio-raw | --pacbio-corr | --nano-raw |

	     --genome-size SIZE --out-dir PATH
	     [--threads int] [--iterations int] [--min-overlap int]
	     [--meta] [--plasmids] [--no-trestle] [--debug]
	     [--version] [--help] [--resume]

```
